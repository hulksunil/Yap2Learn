# Yap2Learn
Learning to Yap

## Pitch
### The Problem
In Montreal, people live bilingually, but learning French or English breaks down in real conversations. Apps teach words and rules, not real-life speaking, and practicing with people is stressful. Existing tools don’t sound or act like real humans.

### Our Solution
We built a voice-first, scenario-based language coach for English and French. Users pick from a list of real-life scenarios and have a natural spoken conversation with an AI that plays a realistic role. It provides real-time, in-context feedback without disrupting the conversation.
